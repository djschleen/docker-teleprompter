# docker-teleprompter

A teleprompter application hosted in a docker container. Enter your own text and away you go.

![](teleprompter.png)



## Usage

Modify the ```config.env``` and ```deploy.env``` to suit your needs from a version and repo perspective.

To build: ```make build```
To run:	```make run```
